📘 College Management System (CMS)
📌 Project Overview

The College Management System (CMS) is a web-based application designed to automate and manage academic and administrative activities within a college.
The system replaces traditional manual record-keeping with a centralized digital platform.

It allows Admin, Faculty, and Students to interact with the system based on role-based access control.

🎯 Objectives

Automate student and faculty record management

Manage courses and academic information efficiently

Maintain attendance and examination records

Provide secure role-based access

Reduce manual paperwork and human errors

Improve operational efficiency

👥 User Roles
🔹 Admin

Add, update, delete students

Add, update, delete faculty

Manage courses

View all records

🔹 Faculty

Manage attendance

Upload exam results

View assigned students

🔹 Student

View attendance

View results

View personal academic details
